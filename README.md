# Chaoxing-All-in-One-Task-Quick-Overview（学习通作业考试全能速览）
这是一个适配学习通的脚本，主要功能：
1. 在学习通首页导航栏新增「全部作业」「全部考试」快捷按钮
2. 点击按钮可查看对应的任务列表，支持搜索筛选作业/考试
3. 版本1.0已修复页面空白的显示异常问题


## 安装方式
### 方法1：ScriptCat（推荐）
[前往ScriptCat安装本脚本](https://scriptcat.org/zh-CN/script-show-page/4807)

### 方法2：油猴插件（Tampermonkey/Violentmonkey）
1. 安装油猴类插件（如Tampermonkey）；
2. 复制仓库中`Chaoxing-All-in-One-Task-Quick-Overview.js`的全部代码；
3. 打开油猴插件→新建脚本→粘贴代码→保存启用。


## 使用教程
1. 安装脚本后，访问学习通首页（`https://i.chaoxing.com/base`）；
2. 左侧导航栏会自动新增「全部作业」「全部考试」快捷按钮；
3. 点击对应按钮，即可查看整理后的作业/考试列表（支持搜索、筛选状态）。
